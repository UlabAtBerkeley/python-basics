1. Open a terminal session in this directory
2. Activate the ulab_3.6 environment
3. Run the command 'jupyter notebook python-basics.ipynb'. This will open a jupyter notebook in your default web browser.
4. To exit your jupyter notebook session, go to "File -> Close and Halt", exit the page, then close your terminal session.

In the future all assignments will consist of a jupyter notebook which you must fill out and submit.
